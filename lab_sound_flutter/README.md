# lab_sound_flutter
**WIP**

lab_sound_flutter is the [LabSound](https://github.com/LabSound/LabSound) wrapper for Flutter.

## Platform
* [x] Android
* [x] iOS
* [x] Mac
* [x] Windows
* [x] Linux
* [ ] Web
